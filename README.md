# Excel Sales Dashboard

## Project Overview
Interactive Excel Dashboard built using Pivot Tables, Pivot Charts, Slicers, Dropdown Lists, and VBA.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dropdown Lists
- VBA

## Features
- Interactive Slicer Dashboard
- Dynamic Dropdown Dashboard
- KPI Cards
- Dashboard Navigation Buttons
